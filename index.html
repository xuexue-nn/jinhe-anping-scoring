設定同步（僅管理員可見）

方式一（推薦）：設定好雲端同步網址後，你在這裡按「同步到雲端」，大家的裝置只要有連網，打開系統就會自動抓到最新設定，不用手動操作。設定步驟見附上的「雲端同步設定教學」。
方式二（備用，沒設定雲端或沒網路時可用）：匯出設定檔案，自己傳給大家（例如透過 LINE），請每個人到「考官評分」分頁最下方手動匯入。

資料合併

方式一（推薦）：如果大家的裝置有設定同步網址，考官評分／自評／投票在儲存或送出時，會自動背景上傳到雲端。你只要按下方「從雲端收集大家的評分與投票」，就能把目前雲端上所有人已經填好的資料一次合併進來，不用再收 JSON 檔。

方式二（備用，沒設定雲端或沒網路時可用）：考官、夥伴、投票人分別在不同電腦 / 手機上填寫，各自「匯出」後把 JSON 檔傳給你，在這裡一次匯入即可自動合併（評分與投票都會自動分類）。
